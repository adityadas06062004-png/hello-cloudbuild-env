# Personal Daily Assistant

A simple and clean personal daily assistant to help you manage your day, track tasks, set reminders, and make critical decisions.

## Features

-   **Task Management:** Add, complete, and delete your daily tasks.
-   **Reminders:** Set reminders with different priority levels (Low, Medium, Urgent).
-   **Decision Helper:** A tool to help you think through decisions by listing pros and cons.
-   **Themes:** Switch between light and dark mode for your comfort.

## How to Use

1.  Clone or download this project.
2.  Navigate to the `personal-assistant` directory.
3.  Open the `index.html` file in your favorite web browser.

That's it! You can now start managing your day.
